# Attention Training Models

This directory stores local outputs from:

- TF-IDF task-head training
- MiniLM embedding-head training
- transformer fine-tuning
- masked-language-model pretraining

Large model files and checkpoints are intentionally excluded from the GitHub upload package.

Reproduce them with:

- `attention_training/train_tfidf_task_heads.py`
- `attention_training/train_embedding_heads.py`
- `attention_training/train_transformer_finetune.py`
